忽然之间，XMind 连插入图片都变成收费功能了，只能默默掉头了。

开源、跨平台、可完美导出 SVG 的 FreeMind 和 FreePlane 一脉相承，其实是相爱相杀，两个都用了一下，对比功能、美观、体验等，推荐 Freeplane。

Ubuntu 安装：

```
$ sudo apt install freeplane
```
